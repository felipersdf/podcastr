## 🧭 Iniciando a aplicação

Para iniciar a aplicação:

```bash
cd podcastr
yarn install // npm install
yarn dev // npm dev
```

Depois executar os passos, uma página com `localhost: 3000` contendo a aplicação desenvolida irá abrir no seu navegador.
